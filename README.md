# builder-catalogue
A helper tool that analyzes your element inventory and shows which sets you can build.

## Features
- Fetch all users and all sets from the API  
- Select a user and explore their inventory  
- List all users or all sets  
- Find sets the user can fully build  
- Find collaborators who can help build a target set  
- Check missing pieces required for a set  
- Calculate the percentage of a set the user can build  
- Simple and interactive CLI interface  

## How to run
1. Install Python
2. Install the requirements 
`pip install -r requirements.txt`
    - requests
    - pytest (for testing)
3. Run the CLI Program
`python main.py`
